Arguments

:: String

Takes the String value returned from the API

-> Maybe String

Returns either Nothing (if the value is "unknown"), otherwise returns the value itself

Converts values of "unknown" to type Nothing

It should only be used if the JSON value is sometimes "unknown" - do not use for Null values

It takes a single argument
